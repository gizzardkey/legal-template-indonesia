# Template Guide

Panduan ini menjelaskan standar penulisan template dalam repository ini.

## Format File

Semua template menggunakan format Markdown (`.md`).

Contoh nama file:

```text
perjanjian-kerja-sama.md
surat-kuasa.md
kontrak-kerja-karyawan.md
```

## Placeholder

Gunakan placeholder dengan format huruf kapital di dalam tanda kurung siku.

Contoh:

```md
[NAMA]
[ALAMAT]
[TANGGAL]
[KOTA]
[NIK]
```

## Struktur Template

Template sebaiknya memiliki:

1. Judul dokumen
2. Identitas para pihak
3. Isi utama dokumen
4. Ketentuan tambahan jika diperlukan
5. Tempat dan tanggal
6. Tanda tangan

## Contoh Struktur

```md
# SURAT PERNYATAAN

Saya yang bertanda tangan di bawah ini:

Nama: [NAMA]

NIK: [NIK]

Alamat: [ALAMAT]

Menyatakan bahwa:

[ISI_PERNYATAAN]

[KOTA], [TANGGAL]

Yang Membuat Pernyataan

(...................)

[NAMA]
```

## Best Practices

* Gunakan Bahasa Indonesia formal
* Hindari bahasa yang ambigu
* Gunakan placeholder yang konsisten
* Jangan memasukkan data pribadi nyata
* Tambahkan catatan jika template membutuhkan penyesuaian khusus
