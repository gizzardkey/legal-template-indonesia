# Legal Template Indonesia

Kumpulan template dokumen legal berbahasa Indonesia yang terbuka, mudah digunakan, dan dapat dikembangkan oleh komunitas.

## Fitur

- Template kontrak kerja
- NDA / perjanjian kerahasiaan
- Surat kuasa
- Perjanjian kerja sama
- Surat pernyataan
- Generator dokumen otomatis
- Format Markdown yang mudah diedit

## Disclaimer

Repository ini hanya menyediakan template umum dan bukan pengganti nasihat hukum profesional.
